# OpenAdBlock — Lightweight ad blocker designed to stay on

A lightweight, open-source ad blocker with breakage-free filters, built for fast and stable browsing.

## MV3

The Chrome/Chromium Manifest V3 implementation is in `mv3/`.

To load the current development build:

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Choose Load unpacked.
4. Select this repo's `chrome/mv3` folder.

Validate extension files with:

```sh
cd mv3
npm run validate
```
